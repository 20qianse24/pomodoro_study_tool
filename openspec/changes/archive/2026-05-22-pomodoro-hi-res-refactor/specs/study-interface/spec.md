## MODIFIED Requirements

### Requirement: Minimalist Layout
The system SHALL feature a centered layout with a modern, focused 'study vibe', incorporating the new time selection and history components.

#### Scenario: Visual presentation
- **WHEN** the user views the application
- **THEN** all core elements (HH:MM:SS Timer, Scroll Selector, Controls, History Button) SHALL be centered on the screen with consistent spacing.

## ADDED Requirements

### Requirement: History Toggle
The system SHALL provide a "History" button to access past session data.

#### Scenario: Opening history
- **WHEN** the user clicks the "History" button
- **THEN** the system SHALL display a modal or overlay containing the session history log.
